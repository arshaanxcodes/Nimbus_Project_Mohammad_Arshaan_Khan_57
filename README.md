# Simple Cyber Incident Tracker

A C-language based console application to log, monitor, and report cyber-security incidents.

> Project Objective

The objective of this project is to build a Simple Cyber Incident Tracking System using the foundational concepts of C programming.
The system allows users to:

1) Add new cyber incidents

2) View all existing incidents

3) Update the age of incidents

4) Automatically escalate old incidents

5) Generate reports (Open / Closed / Escalated)


# Usage Instructions

1. Add Incident

Enter details like ID, type, severity, assigned staff, and status.

2. List Incidents

Displays all incidents in a readable format.

3. Update Ages

Increase the age (in hours) of all incidents.

4. Auto Escalate

Automatically changes the status to "escalated" if age > 48 hours.
